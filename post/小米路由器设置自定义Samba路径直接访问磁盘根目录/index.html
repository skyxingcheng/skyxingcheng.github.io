小米路由器，设置自定义Samba路径，直接访问磁盘根目录

2020年8月30日
| linux samba 路由器
| 阅读

确保您已开启SSH

确保您的外设正常使用且Samba插件可正常开启

首先关闭Samba插件

使用scp登录您的路由器

修改配置文件\extdisks\sda1\xiaomi_router\config\smb.conf

追加代码

1
2
3
4
5
6
7
8

[root]
	path = /extdisks/sda1
	read only = no
	guest ok = yes
	create mask = 0666
	directory mask = 0777
	browseable = yes
	force user = root

修改服务配置\extdisks\sda1\xiaomi_router\init\001-samba(请先备份这个文件) 删除图示模块 你的文件最终应该是这样的

手机APP重新打开Samba插件

enjoy :)
