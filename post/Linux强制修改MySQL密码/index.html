Linux强制修改MySQL密码

2021年4月1日
| linux mysql
| 阅读

一、停掉mysql服务

service mysqld stop

二、启动mysql，禁止权限验证

#在mysql安装目录的bin目录下执行(&后台执行)
mysqld_safe --skip-grant-tables &

三、本地登录，更改密码

本地 localhost登录无需密码

mysql -u root
#切换数据库
use mysql;
#更改密码
update user set password=password("newpasswd") where user="root";
#刷新权限
flush privileges;
#退出
exit;

四、启动mysql 先停掉已经跳过权限的mysql启动进程

#查询mysql是否启动
ps aux | grep "mysql"
#如果发现 有  --skip-grant-tables  类似的进程 找到其 PID kill掉
#PID : 用户名后面的 数字为PID
kill -9 PID
正常启动mysql

service mysqld start

使用设置的密码登录即可

mysql -uroot -pnewpasswd

End
