树莓派折腾日记#1

2020年4月29日
| Raspiberry linux Debian
| 阅读

boot分区下新建文件 ssh 来启用ssh

boot分区编辑config.txt最后一行加上

1

arm_64bit=1

来启用64位内核

使用 pi raspberry 登陆系统

修改密码并解锁root账户

1
2
3

sudo passwd pi				#修改pi的密码
sudo passwd root			#修改root的密码
sudo passwd --unlock root	#解锁root用户

然后使用root登陆 备份镜像源

1
2

cp /etc/apt/sources.list /etc/apt/sources.list.bak
cp /etc/apt/sources.list.d/raspi.list /etc/apt/sources.list.d/raspi.list.bak

修改软件源

1

sudo nano /etc/apt/sources.list

阿里云地址

1
2

deb http://mirrors.aliyun.com/raspbian/raspbian/ buster main non-free contrib
deb-src http://mirrors.aliyun.com/raspbian/raspbian/ buster main non-free contrib

Ctrl+O保存，Ctrl+X退出

修改系统源

1

sudo nano /etc/apt/sources.list.d/raspi.list

还是阿里云

1
2

deb http://mirrors.aliyun.com/raspbian/raspbian/ buster main
deb-src http://mirrors.aliyun.com/raspbian/raspbian/ buster main

Ctrl+O保存，Ctrl+X退出

1

apt-get update && apt-get upgrade -y

1

apt-get -y install vim #先来个vim舒服一下
