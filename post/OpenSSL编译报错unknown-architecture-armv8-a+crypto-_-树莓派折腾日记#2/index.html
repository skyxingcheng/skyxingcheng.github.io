OpenSSL编译报错：unknown architecture `armv8-a+crypto' _ 树莓派折腾日记#2

2020年4月29日
| raspiberry linux debian
| 阅读

RT

先说明，本人小白，搞树莓派的OpenSSL头都大了 这特喵是什么神奇的报错

1

unknown architecture `armv8-a+crypto'

?????OpenSSL不支持armv8指令集???? 解决如下

1
2
3
4
5
6
7
8

cd /usr/src
wget https://www.openssl.org/source/openssl-1.1.1g.tar.gz
tar -zvxf openssl-1.1.1g.tar.gz
cd openssl-1.1.1g
./config no-asm shared zlib --prefix=/usr/local/openssl
make depend
make #别test了 慢的要死
make install
